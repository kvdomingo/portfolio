For this activity [[1](#Soriano2019)], I used the features extracted from the fruits (50 each of apples, mangoes,
bananas) from the [previous activity](/svip/ap186/feature-extract). The feature space in $a^*$-$b^*$ (obtained from the
$L^*a^*b^*$ color space) is shown in Fig.&nbsp;[1](#fig-ab-space). Since we'll be working with a linear classifier for
now, we need to process only two classes at a time. We design the perceptron so that it follows a simple weight update
rule

<span id="eq-weight-update" class="mx-auto">
$
\begin{equation}
  \Delta w_j = \eta (y^i - z^i) x_j^i
\end{equation}
$
</span>

where $y$ is the ground truth label, $z$ is the predicted label, and $\eta$ is the learning rate, which we set to
10<sup>-2</sup>. The perceptron is trained for 100 epochs or until the sum of squares error (SSE) drops below some
selected tolerance $\epsilon = 10^{-6}$ and the decision boundary is obtained from the final weights. The decision
boundary and decision contours for each class pair is shown in Fig.&nbsp;[2](#fig-boundaries).
